If you want to increase the distance between anything, like the table above and this parapgrah add the padding class instead of br. In this example this paragraph has padding added to the top with the class pt-4. p being padding, t being top (change to b for bottom padding) and 4 for the size. You can have padding 1 to 5 with this grid system.

For longer tables add the class table-striped to the table tag to aid readability. This is why having the thead and tbody is important. I would say anything with more than 6 rows might need this class.

A few tips:

  • Write your closing tag immediately after writing your opening tag. Then write any content between.
  • Adding comments to your div pairs can help you work out if you've missed closing any. Indents help too.
  • Write your content first with basic tags, p, h1, h2, img ...ect and add any styling (inlcuding spacing) afterwards. You might not need any when you see it as a whole.
  • You don't need to add anchor tags with IDs. You can attach the ID to any element like an h2 or p tag. You only need an empty anchor tag if you need to target a section of the page that has no other tags to use.
  • Don't add brs inside paragraphs.
  • Tables and lists aren't allowed inside paragraphs.
