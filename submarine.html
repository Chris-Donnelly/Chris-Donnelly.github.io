Real-time underwater 3D graphics Scene (using Direct3D 11) with dynamic lighting (multiple types), PCF shadowing, and behavioural patterns.

Gravity Wells Simulation

Written in C++11 (Visual Studio 2015), using Win32, Direct3D 11, GLM and AntTweakBar

MSC Computer Science for Video Game Development, University Of Hull, UK

Context

A real-time 3D shaded graphics scene, using lighting (global lighting, spotlights) and shadows (PCF), set underwater. The scene features two submarines of different types, with animated behaviours (drebbel and nuclear submarine), animated moving sea life (fish), encased in a 'sphere' or 'bubble'. The program also features postprocessing shader effects (monochrome, sepia, toon), time scaling and exponential fog.
